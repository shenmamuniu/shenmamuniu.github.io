CentOS7安装MariaDB

MariaDB
数据库
发布日期:   2021-04-20
更新日期:   2021-04-20
文章字数:   525
阅读时长:   2 分

最近在做数据迁移，顺便把mysql替换为mariadb.

环境

OS:centos 7.6
Mariadb：10.5.9

提示:关于mariadb的安装，网上有太多乱七八糟的教程了，最好的办法还是参考mariadb官网文档。

软件下载

wget https://mirrors.cloud.tencent.com/mariadb/mariadb-10.5.9/bintar-linux-systemd-x86_64/mariadb-10.5.9-linux-systemd-x86_64.tar.gz

删除已安装的mysql or mariadb

这里不仅需要移除mariadb，如果有mysql,也要移除

rpm -qa | grep mariadb
rpm -ev mariadb-libs-5.5.68-1.el7.x86_64 --nodeps
find / -name mysql # 删除查询出来的文件夹

创建用户和组

groupadd mysql
useradd -r -g mysql mysql
chown -R mysql:mysql /usr/local/mysql

创建数据目录

chown -R mysql:mysql /opt/data/mariadb

配置数据库的配置文件

mariadb默认使用的配置文件/etc/my.cnf

[client]
port = 3306
socket = /usr/local/mysql/mysqld.sock

# The MySQL server
[mysqld]
port = 3306
socket = /usr/local/mysql/mysqld.sock
datadir = /opt/data/mariadb
skip-external-locking
skip-name-resolve
back_log = 50
max_connections = 2048
max_connect_errors = 1000
table_open_cache = 1024
open_files_limit = 16384
max_allowed_packet = 16M
read_buffer_size = 8M
read_rnd_buffer_size = 32M
sort_buffer_size = 2M
join_buffer_size = 2M
thread_cache_size = 64
query_cache_size = 64M
query_cache_limit = 4M
slow_query_log = 1
long_query_time = 2
lower_case_table_names = 1
innodb_file_per_table = 1
max_allowed_packet = 1G
server-id = 1
log-bin = mysql-bin
expire_logs_days = 7
binlog_format = ROW

innodb_data_file_path = ibdata1:12M:autoextend
innodb_buffer_pool_size = 2G
innodb_write_io_threads = 12
innodb_read_io_threads = 8
innodb_thread_concurrency = 16
innodb_flush_log_at_trx_commit = 2
innodb_log_buffer_size = 16M
innodb_log_file_size = 170M
innodb_lock_wait_timeout = 60

数据库初始化

./scripts/mysql_install_db --user=mysql

# 启动
./bin/mysqld_safe --user=mysql &

配置开机启动

cp /usr/local/msyql/support-files/mysql.server /etc/init.d/mysql
chmod 755 /etc/init.d/mysql
chkconfig --add mysql
chkconfig --level 345 mysql on

设置环境变量

export PATH=/usr/local/mysql/bin:$PATH

service mysql start/restart

设置用户

mysql>grant all privileges on *.* to 'root'@'%' identified by 'root' WITH GRANT OPTION;;
mysql>flush privileges;

这里注意添加with grant option，通过update会提示没有权限.
到这一步，就可以通过mysql client连接数据库了。
