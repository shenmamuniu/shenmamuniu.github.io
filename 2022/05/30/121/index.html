基于PostGIS的等值线分析服务发布

Python postgis
Python
发布日期:   2022-05-30
更新日期:   2022-05-30
文章字数:   712
阅读时长:   3 分

在用Cesium原生的等高线工具时，由于地形存在分级，不同级别下的地形的精度不同，因此生成的等高线也在时刻发生变化，且无法知道高程，给用户造成了一定的困惑。正好PostGIS3.2中提供了ST_Contour函数可生成等高线，本文即利用ST_Contour函数来生成等值线。

应用环境

PostGIS:3.2.1

Postgresql:14.3

OS:win10

python:3.9

地形导入PostGIS数据库

QGIS只提供矢量数据的导入，并未提供栅格数据的导入功能，这里可使用PostGIS安装目录下的raster2pgsql程序进行导入，命令如下：

./raster2pgsql.exe -s 4326 -C E:/Data/dem/江津区_DEM.tif public.dem | ./psql.exe -h localhost -p 5432 -U postgres -d chongqing -W

关于参数说明，可通过以下命令查看

./raster2pgsql.exe

提示:不能使用SuperMap IDesktop导入栅格数据，导入后的格式不太一样，会导致后面的ST_Contour函数无法使用。

使用ST_Contour函数

使用如下sql语句进行查询:

WITH c AS (
SELECT (ST_Contour(rast, 1, fixed_levels => ARRAY[1000.0, 1200.0, 1300.0])).*
FROM public."dem" WHERE rid = 1
)
SELECT id, value,st_astext(geom)
FROM c order by value asc;

ST_Contour函数的参数说明:

参数名 说明
level_interval 等值线间隔
fixed_levels 取固定的高程值

Python连接PostGIS提供REST服务

这里使用psycopg2连接postgis数据库,使用tornado提供REST服务。

安装psycopg2及tornado:

pip install psycopg2 -i https://mirrors.aliyun.com/pypi/simple/
pip install tornado -i https://mirrors.aliyun.com/pypi/simple/

也可以下载whl文件离线安装:

pip install xx.whl

psycopg2的使用可参考,代码如下：

import psycopg2
conn = psycopg2.connect(database='postgis', user='postgres')
curs = conn.cursor()

# Find the distance within 1 km of point-of-interest
poi = (-124.3, 53.2)  # longitude, latitude

# Table 'my_points' has a geography column 'geog'
curs.execute("""\
SELECT gid, ST_AsGeoJSON(geog), ST_Distance(geog, poi)
FROM my_points, (SELECT ST_MakePoint(%s, %s)::geography AS poi) AS f
WHERE ST_DWithin(geog, poi, 1000);""", poi)

for row in curs.fetchall():
    print(row)

小结

这种方法在查询速度和准确性上相比其它方法都有明显的提升，但线段之间的平滑度不够，后续还需要在这方面进行优化。
