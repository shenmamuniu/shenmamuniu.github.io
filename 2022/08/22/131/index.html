Python实现将Markdown转PDF

Python

由于Markdown在移动端读取稍微困难，因此存在将markdown文件转为PDF文档的需要。首先将markdown文件转为html,然后再用工具将html转为pdf文件。

应用环境

OS:windows
Python:3.9

文件遍历

# md文件路径
original_path = u'xxx/xxx' 
# 输出路径
saved_path = u'xxx/xxx'

for root,dirs,files in os.walk(original_path):
    # for dir in dirs:
    #     print(os.path.join(root,dir))        
    for file in files:
        if file.endswith('md'):
            input=os.path.join(root,file)
            file = file[:-2] + 'pdf'
            dirName=''
            # 处理多级目录的情况
            if root==original_path:
                dirName=root.replace(original_path,'')
            else:
                dirName=root.replace(original_path+'\\','')
            # print(dirName)
            dirOpt = os.path.join(saved_path, dirName)
            if not os.path.exists(dirOpt):
                # 相比mkdir,此方法可创建多级目录
                os.makedirs(dirOpt)
            output=os.path.join(dirOpt,file)

            if not os.path.exists(output):
                cvtMd2Pdf(input=input,output=output)

markdown转pdf

import pdfkit
from markdown import markdown
import os

def cvtMd2Pdf(input,output):
    with open(input, encoding='utf-8') as f:
        text = f.read()

    html = markdown(text, output_format='html',extensions=['tables'])  # MarkDown转HTML
    htmltopdf = r'C:\\Program Files\\wkhtmltopdf\\bin\\wkhtmltopdf.exe'
    configuration = pdfkit.configuration(wkhtmltopdf=htmltopdf)
    pdfkit.from_string(html, output_path=output, configuration=configuration, options={'encoding': 'utf-8'})

缺点和不足

1、存在某些文件转换失败的问题
2、转换效率不高
