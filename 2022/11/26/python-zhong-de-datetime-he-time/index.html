Python中的datetime和time

Python
Python

阅读时长:   2 分

本文记录Python中的日期和时间的用法。

应用环境

Python:3.9

OS:Windows

IDE:VS Code

时间：time

本节内容来源：Python3 日期和时间

获取当前时间

import time
print (time.strftime("%Y-%m-%d %H:%M:%S", time.localtime()))

输出：

2022-11-26 15:43:44

将时间字符串转为时间

这里有个struct_time的东西，为tuple格式：

time_struct=time.strptime("2022-11-26 15:43:44","%Y-%m-%d %H:%M:%S")
print(time_struct)

输出结果如下:

time.struct_time(tm_year=2022, tm_mon=11, tm_mday=26, tm_hour=15, tm_min=43, tm_sec=44, tm_wday=5, tm_yday=330,
tm_isdst=-1)

注：这个是可以直接使用>,<等运算符直接比较大小的

如判断当前时间是否在某个时间段：

def checkTimePassed(startTime,endTime):
    start=time.strptime(startTime,"%Y-%m-%d %H:%M:%S")
    end=time.strptime(endTime,"%Y-%m-%d %H:%M:%S")
    nowstr=datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    now=time.strptime(nowstr,"%Y-%m-%d %H:%M:%S")
    if now>start and now<end:
        return True
    else:
        return False

将time_struct转为时间戳：

print (time.mktime(time.strptime("2022-11-26 15:43:44","%Y-%m-%d %H:%M:%S")))

输出

1669448624.0

时间格式化字符串

  • %y 两位数的年份表示（00-99）
  • %Y 四位数的年份表示（000-9999）
  • %m 月份（01-12）
  • %d 月内中的一天（0-31）
  • %H 24小时制小时数（0-23）
  • %I 12小时制小时数（01-12）
  • %M 分钟数（00=59）
  • %S 秒（00-59）

datetime

获取当前时间

import datetime
dt=datetime.datetime.now()
print(dt.strftime('%Y-%m-%d %H:%M:%S'))

输出

2022-11-26 15:50:27

获取前10分钟的时间

使用datetime的timedelta方法：

import datetime
dt=datetime.datetime.now()
print(dt.strftime('%Y-%m-%d %H:%M:%S'))
yes_time = dt + datetime.timedelta(minutes=-10)
newTime=yes_time.strftime('%Y-%m-%d %H:%M:%S')
print(newTime)

输出：

2022-11-26 15:51:15
2022-11-26 15:41:15

strftime vs. strptime

strftime:将时间类型转为指定格式的字符串

strptime:将时间字符串转为时间
