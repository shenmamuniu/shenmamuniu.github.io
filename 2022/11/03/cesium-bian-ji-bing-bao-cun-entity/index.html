Cesium编辑并保存Entity

Cesium KML Geojson
Cesium
发布日期:   2022-11-03
更新日期:   2022-11-05
文章字数:   160
阅读时长:   1 分

这里借助Cesium的exportKml方法，先将Entity转为xml字符串，然后借助mapbox的togeojson将kml转为geojson,从而实现Entity数据的保存。

Cesium编辑Entity

可参考绘制编辑线面

Cesium将Entity转为kml

参考exportKml:

let entities = new Cesium.EntityCollection();
entities.add(x);//其中x为Entity
Cesium.exportKml({
     entities: entityCollection
 })
  .then(function(result) {
    var xmlStr=result.kml;
  });

kml转geojson

使用togeojson可将kml转为geojson,如下:

<script src='togeojson.js'></script>
<script>
$.ajax('test/data/linestring.kml').done(function(xml) {
    console.log(toGeoJSON.kml(xml));
});
</script>

如果拿到的是xml的字符串形式，即：

"<xml></xml>"

需要先将xml字符串格式化，再调用toGeoJSON.kml函数:

var dom = (new DOMParser()).parseFromString(xmlStr, 'text/xml');
